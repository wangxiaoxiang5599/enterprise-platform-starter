# Enterprise Platform Starter

Enterprise-grade backend starter for building SaaS and AI-powered applications.

## Features
- JWT Authentication
- RBAC (User / Role / Permission)
- Multi-tenant architecture
- Data scope control (ALL / DEPT / SELF)
- Audit logging (who did what, when, and how)
- Structured logging & traceId
- Docker-compose ready

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Security 6
- PostgreSQL
- Redis
- Docker
